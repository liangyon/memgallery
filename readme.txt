Hello, and welcome to photolibrary!


In this application, you can create a secure account, and manage a secure collection of photos for each account.

Included features:
- Personal Gallery System
- Image upload System
- User System
- Image information System


To use: clone onto your device
make sure django is installed
py manage.py runserver

run and access from http://127.0.0.1:8000/photolibrary/
